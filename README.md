# Django Legado

Este projeto foi desenvolvido com o objetivo de **estudar a estrutura e manutenção de aplicações Django baseadas em código legado**, analisando como sistemas existentes podem ser compreendidos, mantidos e evoluídos.

Em muitos ambientes corporativos, grande parte dos sistemas já está em produção há anos. Esses sistemas frequentemente possuem **código legado**, que precisa ser compreendido antes de qualquer melhoria ou refatoração.

O objetivo deste repositório é explorar **como analisar, executar e adaptar um projeto Django legado**, entendendo sua arquitetura, organização e dependências.

---

# 🎯 Objetivo do projeto

Estudar conceitos relacionados a **manutenção e análise de sistemas existentes**, incluindo:

- Estrutura de um projeto Django legado
- Organização de apps e módulos
- Entendimento do fluxo da aplicação
- Análise de dependências
- Execução e manutenção de projetos existentes
- Preparação para refatoração futura

---

# 🧰 Tecnologias utilizadas

- Python
- Django
- SQLite
- HTML / Templates Django

O **Django** é um framework web de alto nível escrito em Python, projetado para acelerar o desenvolvimento de aplicações web seguras e escaláveis.
